Configuration & Administration

Gestion des utilisateurs et déploiement du service Web Apache2.

1. Environnement Système

Environnement graphique GNOME fonctionnel.

Premier Démarrage

Après l'installation des "Invités Additions" (Guest Additions) pour gérer l'affichage plein écran, nous accédons à notre environnement de travail. Le système est stable et prêt pour la configuration.

Gestion des Utilisateurs

Nous avons structuré les accès en créant un groupe dédié au projet.

Commandes utilisées :
  • sudo addgroup Trio04
    Création du groupe de travail.
  • sudo adduser admin_Gr107_Trio04
    Création du compte administrateur principal.
  • sudo usermod -aG Trio04 Trio04_1
    Ajout des utilisateurs (membres du trinôme) au groupe.

2. Serveur Web (Apache)

Pour héberger ce rapport, nous avons choisi le serveur HTTP Apache2, une référence pour sa robustesse sous Linux.

Installation

Mise à jour des dépôts puis installation du paquet :
sudo apt install apache2

Configuration

Modification de la page d'accueil par défaut pour tester les droits d'écriture dans /var/www/html/.

Résultat

Succès ! Le serveur répond sur le port 80 via l'adresse localhost.
