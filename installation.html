Installation de la Machine Virtuelle

Tutoriel pas-à-pas pour la création du poste de travail Ubuntu.

1. Création de la VM

Dans VirtualBox, nous créons une nouvelle machine en cliquant sur "Nouvelle".

  • Nom : Ubuntu (pour identifier la distribution).
  • ISO Image : Sélection de ubuntu-24.04.3-desktop-amd64.iso.
  • Type : Linux / Ubuntu (64-bit).
Note : Nous cochons l'option pour installer manuellement l'OS ("Skip unattended install") afin de maîtriser le partitionnement.

2. Allocation des Ressources

Pour supporter l'environnement graphique GNOME et l'IDE WebStorm, nous allouons des ressources conséquentes :

  • Mémoire RAM : 4096 MB (4 Go). Indispensable pour la fluidité.
  • Processeurs (CPU) : 2 cœurs. Permet de gérer le multitâche sans ralentir l'hôte.
  • Disque Dur : 30 Go minimum (Alloué dynamiquement).

3. Premier Démarrage (Boot)

Une fois la configuration terminée, nous lançons la machine. Le menu GNU GRUB apparaît.

Nous sélectionnons l'option :
Try or Install Ubuntu

Le système va charger les fichiers d'installation en mémoire.

4. Installation du Système

Après le choix de la langue (Français) et du clavier, nous arrivons au partitionnement.

Choix technique :

Option choisie : "Effacer le disque et installer Ubuntu".

Comme il s'agit d'un disque virtuel (VDI) vide, cette option est sûre. Elle formate le disque virtuel pour préparer le système de fichiers Linux (ext4).
