Le Projet

L'objectif de cette SAE est d'installer et de configurer un poste de travail complet basé sur le système d'exploitation Ubuntu 24.04 LTS.

Nous avons mis en place une machine virtuelle capable de supporter des outils de développement comme PhpStorm et un serveur web Apache2. Ce site web fait office de rapport technique et documente l'intégralité de notre démarche.

Informations Clés
  • OS : Linux Ubuntu 24.04 (64-bit)
  • Virtualisation : VirtualBox (Type 2)
  • RAM allouée : 4096 Mo
  • Serveur Web : Apache 2
